# منصة الذكاء الاصطناعي لدراسة المشاريع والمقاولات

تطبيق ويب عربي RTL لإدارة وتحليل مشاريع المقاولات وتوليد دراسة جدوى أولية، خطة تنفيذ PMP، جدول كميات، مراكز تكلفة، مخاطر، وتقارير قابلة للتصدير.

## التشغيل

لا يحتاج التطبيق إلى حزم خارجية في وضع العرض التجريبي، ويعمل مباشرة باستخدام Node.js 18 أو أحدث:

```bash
npm install
npm start
```

ثم افتح المتصفح على:

```text
http://localhost:3000
```

بيانات الدخول التجريبية:

- البريد: `admin@erp.local`
- كلمة المرور: `admin123`

## الوحدات

- لوحة التحكم والمؤشرات.
- إنشاء مشروع بالذكاء الاصطناعي.
- تفاصيل المشروع وخطة PMP.
- دراسة الجدوى.
- التكاليف ومراكز التكلفة.
- المخاطر.
- التقارير والتصدير.

## قاعدة البيانات

يوجد مخطط PostgreSQL اختياري في `database/schema.sql` يتضمن الحقول المطلوبة لكيان `projects`. الخادم الحالي يعمل ببيانات تجريبية داخل الذاكرة حتى يمكن تشغيله بدون إعدادات خارجية.
